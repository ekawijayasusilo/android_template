Ember

Material 3 design system & screen mockups
A reusable Android (Jetpack Compose) project template · generic social feed demo
lightbulb
Recommended identity — “Ember.” Warm, energetic, one syllable. The mark is an M3 rounded-square badge holding a white speech bubble with a glowing coral dot — built from primitives, so it ships cleanly as an adaptive launcher icon and a Compose ImageVector. Swap the wordmark and you have your own clone.

Each role above is a lightColorScheme() / darkColorScheme() argument. Toggle dynamic color with dynamicColorScheme() on Android 12+.
Typography
One FontFamily (Outfit) across all 15 TextStyle roles — sizes, weights and line-heights as listed.
Shapes & spacing
The corner scale feeds Shapes(...); the 4dp grid drives every padding / Arrangement.spacedBy.
