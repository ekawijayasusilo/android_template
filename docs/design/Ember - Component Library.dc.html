Compose wrapper notes
Hook surface
Every component exposes analyticsId: String? and onImpression: () → Unit as optional lambdas. Wrapper logs them via the DI'd analytics client.
Accessibility
contentDescription is non-nullable on icon buttons. Buttons have minimum minHeight = 48.dp enforced by the wrapper. State announcements wired to LocalAccessibilityManager.
Dynamic color
All tokens are resolved at call-site from MaterialTheme.colorScheme. When dynamic color is on (S+), M3 replaces the coral seed automatically — no component code change needed.
